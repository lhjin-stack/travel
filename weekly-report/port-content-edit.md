## JEA (Jebel Ali)
- 상태: 직항 불가 — KLF 종착

**부킹 가이드**
- B/L 기재: POD/DLY 모두 KLF (JEA 기재 금지)
- KLF→JEA: 보세운송 IN-TRANSIT 처리 (DPW Logistics 계약, CNEE 비용 부담)
- KLF 통관 희망 시 → 선박 도착 2일 전 사전 통보 필수 (미통보 시 JEA 자동 이송)

**비용**
- DVC (Deviation): $800/20' · $1,600/40'
- TRC (트럭킹): $900~1,000/VAN (2일마다 변동)
- PRC (DPW Drop-off): AED 700/20' · AED 1,070/40'
- LFT (KLF 스토리지): Day 1부터 청구

**유의사항**
- LOI 징구 필수
- KLF 3일 내 미픽업 → SHJ ICD 강제이송
- 양하 후 5-6일 지연(야적장 부족)
- ICD 이동 후 추적 불가

## DMM / KSP (Dammam / King Salman Port)
- 상태: 직항 불가 — JED 경유 또는 KLF 경유

**루트 1: JED 경유 (권장, 최근 요청 증가)**
- NSA → JED (IRX2 이용, 별도 피더비 없음)
- JED → DMM/KSP: 화주 비용 트럭킹
- 비용:
  - 수입자: DEVIATION $800/$1600 + 도착지 로컬차지 + 로컬 운송비
  - JED→DMM/KSP 트럭킹: 수입자 부담
  - 현지 로컬비용: 화주 진행

**루트 2: KLF 경유**
- NSA → KLF → JEA → DMM/KSP (더블 T/S)
- 피더비 별도 발생

**유의사항**
- LOI 필수
- JED 경유 루트 권장 — 화주 비용 부담 사전 안내 필수

## KWI / UQR / HMA (Upper Gulf 기타)
- 상태: 되도록 Ship-back 권유

**현황**
- 피더 선복·현지 운송·통관 가능성·비용에 따라 별도 검토
- 현재 피더·운송·통관 상황 불투명
- 되도록 Ship-back 권유

**유의사항**
- LOI 필수
- Ship-back 시 송하인·수하인 서면 동의 필수

## SHJ / AJM / RAK / QIW
- 상태: KLF 경유 트럭

**부킹 가이드**
- Routing: POL → KLF → 트럭 → SHJ/AJM/RAK/QIW
- B/L: POD: KLF / DLY: 최종 목적지
- JEA와 동일 기준 적용 | 사전 화주 안내 후 요청 시 진행
- AJM·RAK: Hutchison Logistic 협의 후 처리 (비용 정산 방식 확인 필요)
- 세관 문제 해결 완료 — 보세운송 가능

**비용**
- DEVIATION $800/$1600 + 도착지 로컬차지 + 로컬 운송비

**유의사항**
- LOI 필수
- KLF 3일 내 미픽업 → SHJ ICD 강제이송

## AQJ / JED / SKN (홍해)
- 상태: 더블 T/S — 적체 경보

**부킹 가이드**
- Routing: POL → KLF → 트럭 → JEA → 피더(JJS/IRX2) → AQJ/JED/SKN
- B/L: POD: KLF (TRUCK) → JEA (FEEDER) → DLY
- T/S 신고 후 피더 연결

**비용**
- DEVIATION $800/$1600 + 도착지 로컬차지 + 로컬 운송비
- 피더사 WRS 별도 부과
- 현재 비용 매우 높고 피더 현실적 불가 — 화주에게 반드시 사전 안내

**유의사항**
- JEA·MUN T/S 적체 경보 발령 중 (W11~W15)
- LOI 필수
- 더블 T/S로 운송 기간 대폭 연장

## NSA (Nhava Sheva)
- 상태: 대체 양하항 (EOV)

**처리 가이드**
- AIM2·VGI 중동향 ROB 전량 NSA 양하 결정 (2,050 TEU)
- EOV 선언 화물 대체 양하항
- KLF 도착 이후 처리는 각 DLY 포트별 방안과 동일 적용
- 상세 옵션은 NSA EOV 가이드 탭 참조

**유의사항**
- LOI 필수
- 화주 사전 합의 필요
- 비용·이송방법 현지 상황에 따라 변동 가능
